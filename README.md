# CA Monk Blog Application

A split-view blog application built with **React**, **Vite**, **Tailwind CSS**, **shadcn/ui**, and **TanStack Query**.

## Features
- Left panel: Blog list with instant selection
- Right panel: Blog detail view
- Create Blog modal with form validation
- API state management using TanStack Query
- Loading & error states
- Responsive and clean UI

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS
- shadcn/ui (Radix-based components)
- TanStack Query
- JSON Server (for mock API)

## Setup

```bash
npm install

Start the mock API (assumes JSON Server on port 3000):

npx json-server --watch db.json --port 3000


Run the app:

npm run dev


Open: http://localhost:5173
